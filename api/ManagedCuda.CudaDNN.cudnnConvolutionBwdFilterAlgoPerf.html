    status

    If any error occurs during the workspace allocation or timing of cudnnConvolutionBackwardFilter_v3(), this status will represent that error. Otherwise, this status will be the return status of cudnnConvolutionBackwardFilter_v3().

    • CUDNN_STATUS_ALLOC_FAILED if any error occured during workspace allocation or deallocation.

    • CUDNN_STATUS_EXECUTION_FAILED if any error occured during timing calculations.

    • Otherwise, this will be the return status of cudnnConvolutionBackwardFilter_v3().
